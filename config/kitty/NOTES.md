# Kitty — notas

- No home, o `kitty.conf` costuma ser o template enorme do pacote + tema Catppuccin via `include current-theme.conf`.
- Neste lab mantemos um arquivo **curto** para aprender opções reais.
- Ao linkar, se você quiser preservar o tema atual, copie `~/.config/kitty/current-theme.conf` para o lab ou reative o `include` após o link.

Exercício: mude `background_opacity` e reabra o kitty (`SUPER+Q`).
